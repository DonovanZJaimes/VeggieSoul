
# VeggieSoul
Esta es una aplicación que te lleva a explorar un lugar repleto de recetas veganas. 
Encuentra los ingredientes perfectos para tus platos y perfecciona tus habilidades en la cocina con nuestras prácticas herramientas. 
Además, haz un seguimiento de tu progreso en tu viaje hacia una alimentación más saludable y consciente.


## Separacion de las vistas principales
En esta aplicación, encontrarás cuatro vistas principales. La primera te permitirá descubrir una amplia variedad de recetas vegetarianas, 
la segunda te ayudará a encontrar una diversidad de ingredientes, la tercera ofrecerá una gama de herramientas para facilitarte en la cocina, 
y la cuarta vista te permitirá rastrear tu progreso a medida que avanzas en tu viaje culinario.

### Vista 1: Recetas
En esta vista, tendrás acceso a una amplia gama de recetas vegetarianas. Puedes explorarlas por categoría, buscar recetas que se adapten a 
los ingredientes disponibles o buscar por nombre. Además, tienes la opción de aplicar filtros para encontrar exactamente lo que estás buscando.
También podrás obtener detalles completos de cada receta, incluyendo la lista de ingredientes, los pasos a seguir y su información nutricional. 
Además, tendrás la capacidad de agregar recetas a tu dieta para un acceso rápido y fácil

## Diseño MVC
Esta aplicación sigue el patrón de diseño Modelo-Vista-Controlador (MVC), y en cada carpeta encontrarás normalmente la siguiente estructura de organización de archivos
- View y su ViewController
- Table o CollectionViewCell con sus respectivos archivos Xib
- Vistas complementarias
- Controladores auxiliares
- Modelos

## Spoonacular API
Esta aplicación está siendo desarrollada utilizando la API "Spoonacular API". Esta API ha sido fundamental en cada vista de la aplicación,
facilitando la obtención de recetas, ingredientes, herramientas y la posibilidad de crear dietas personalizadas.
Pagina web de la API: [Spoonacular API](https://spoonacular.com/food-api).
